## MpcBlog

MpcBlog是一个妈妈去哪儿产品部内部使用的轻量级博客应用

### 技术栈

- 前端：Nuxt.js + Vuex
- 后端: Mongoose + Koa

### 功能特性

- 支持服务端渲染
- PWA渐进式web应用
- 轻量级Markdown编辑器
- 支持标签、归档、搜索草稿箱等功能

### 技术支持

感谢[wmui](https://github.com/wmui/vueblog)技术支持。
